# MedRash

> Transforming passive medical presentations into engaging, measurable learning experiences.

## Overview

MedRash is a lightweight, mobile-first gamified medical learning and engagement platform designed for healthcare professionals. It enables pharmaceutical companies, medical representatives, medical affairs teams, and healthcare organizations to transform traditional presentations, CMEs, and educational sessions into short, interactive quiz experiences.

The platform combines learning reinforcement, competitive engagement, and analytics to improve knowledge retention while generating actionable intelligence on product awareness, disease understanding, and educational gaps.

The pilot deployment is focused on healthcare professionals in Ghana and supports both live session participation through QR codes and self-paced learning outside scheduled events.

---

## Key Objectives

### Engagement

Convert passive presentations into interactive experiences that maintain participant attention and improve recall.

### Learning

Reinforce scientific and product knowledge through short, evidence-based quiz interactions.

### Intelligence

Capture measurable data on participant understanding, knowledge gaps, misconceptions, and engagement patterns.

### Scalability

Provide a foundation for future game modes, competitions, CME integrations, and advanced analytics.

## User Journey

### Live Session Flow

1. Participant scans QR code
2. Quick Join completed
3. Silent account created
4. Quiz opens automatically
5. Questions answered
6. Score calculated
7. Results displayed
8. Leaderboard updated (Ranked Mode only)

### Open Access Flow

1. Participant opens MedRash
2. Selects a topic or product
3. Chooses Learning or Ranked Mode
4. Completes quiz
5. Reviews explanations
6. Tracks progress over time

## Game Modes

### Learning Mode

Designed for knowledge improvement.

Features:

* Unlimited attempts
* Score improvement
* Explanations after completion
* Does not affect leaderboard rankings

### Ranked Mode

Designed for competition and benchmarking.

Features:

* One ranked attempt per quiz
* Contributes to leaderboards
* Server-enforced attempt restrictions
* Fair ranking system

## Leaderboards

### All-Time Rankings

Aggregates ranked performance across all eligible quizzes.

### Monthly Rankings

Tracks participant performance within the current leaderboard season.

### Privacy Model

Public rankings display:

* Nickname
* Avatar
* Rank
* Score

Public rankings never display:

* Full Name
* Facility
* Specialty
* Personal Identifiers
